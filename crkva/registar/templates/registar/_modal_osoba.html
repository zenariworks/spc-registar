{% comment %} Quick-add Osoba modal. Uses the generic Modal component (registar/components/modali.css + modal.js). The Modal.bindForm call is deferred to DOMContentLoaded because this partial renders inside {% block content %} while modal.js is loaded at the bottom of base.html. Running Modal.bindForm inline at parse time would hit a ReferenceError (Modal still undefined), so `var osobaModal` would never be assigned. The "+ Додај новог" footer inside Osoba select2 dropdowns checks window.osobaModal before opening the modal, so an unset value made the click a silent no-op on every page that includes this partial. {% endcomment %}
